Composite Project 1

For this Composite Project you are to create an Interface design with a web grid to present a compliant website mock-up before going to the coding phase. This will save you a lot time and communication barriers before the client commit to the deployment of the website. You are going to use Adobe Photoshop to complete the entire website interface assignment with in-class exercises to help you achieve this.

Components to be used:

  • Guides and Grid Setup
  • Vector objects to outline source placements
  • Typography is to be properly implemented to compliment the overall design
  • 1400px width design setup
  • Layers and other elements
  • Images and content are also to be included
